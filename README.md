# Bahire Hasab Engine 🚀

A Golang-based Ethiopian Orthodox calendar engine that calculates fasting days, holidays, and provides a foundation for a spiritual companion app.

---

## 🎯 Features

### ✅ Core Calculations

* Amete Alem
* Wengelawi
* Metene Rabiet
* Tinte Qemer
* Medeb / Wenber
* Abekte / Metqi

### 📅 Calendar Logic

* Beale Metqi
* Tewsak
* Mebaja Hamer
* Nenewe (Nineveh)

### ✝️ Holidays

* Abiy Tsom
* Debre Zeit
* Hosanna
* Siklet
* Fasika

### 🌐 API Endpoints

```
GET /bahire/:year/metqi
GET /bahire/:year/nenewe
GET /bahire/:year/fasika
GET /bahire/:year/all
```
